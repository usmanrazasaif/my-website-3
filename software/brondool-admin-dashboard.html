How it works?

How it works?

Login and Dashboard

The admin logs into theBrondool Admin Dashboard via the login page. The dashboard displays all registered users and the devices for Brondool Lock Security.

User Management

The dashboard has a user-friendly interface for managing users. Admins can add, edit, and delete users, view their status, as well as update their own profile.

Features

Features

User Status Monitoring

Administrators can easily monitor the status of all users, distinguishing between blocked and unblocked accounts at a glance.

User Management

The dashboard provides a user-friendly interface for adding, editing, and deleting users, granting administrators complete control over user profiles.

Admin Profile Editing

Administrators can conveniently update their own profiles within the dashboard, ensuring accurate representation of their information.
